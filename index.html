Яндекс Трекер

Яндекс.Трекер — это система для управления проектами и организации работы в компании, распределения ресурсов и контроля выполнения задач.

Возможности Яндекс.Трекер:

  • Живые задачи в реальном времени.
  • Очереди задач для группировки.
  • Фильтры и поиск.
  • Дашборды, визуализация прогресса.
  • Agile-доски.
  • Шаблоны задач и комментариев.
  • Избранное и подписки.
  • Учёт времени и трудозатрат.
  • Напоминания и призывы.

Цены:

  • 60 дней бесплатно.

Способ оплаты: По подписке

Платформы: Web, iOS, Android

YouTrack

YouTrack — это сервис, разработанный специально для программистов и команд разработки, как и все продукты компании JetBrains. Быстрый и удобный поиск при помощи поисковых запросов, поддержка клавиатурных шорткатов для любых действий, командный синтаксис для редактирования нескольких задач одновременно, гибкая настройка системы под рабочие процессы любой сложности экономит время для всей команды, позволяя сфокусироваться по продукте, а не на процессе разработки. Поддрерживает Scrum и Kanban, локализован на русский язык, предоставляет множество отчетов и осуществляет учет времени.

Функции YouTrack:

  • Комментирование задач.
  • Добавление наблюдателей.
  • Голосование.
  • Уведомления на e-mail или в чат.
  • Встроенное редактирование снимков экрана, добавляемых к задаче или комментарию.
  • 17 видов отчётов.
  • Полнотекстовый поиск.
  • Сохраненные поиски.
  • Метки для группировки.
  • Фильтры для начинающих пользователей.

Цены:

  • Пробный период.
  • Имеется бесплатный план для 10 пользователей (проекты будут публичные).
  • Для приватных проектов сервис стоит от $20 в месяц.

Способ оплаты: По подписке

Платформы: Web, iOS, Android

Планиро

«Планиро» — это гибкое и масштабируемое веб-приложение для удобного управления проектами и совместной работы. Разработчики реализовали модульную систему, предлагая своим клиентам базовую функциональность и модули под её увеличение.

  • Модуль работы с задачами
  • Планирование, контроль и оценка своей работы и работы сотрудников.
  • Работа с любого устройства.
  • Добавление комментариев к задачам.
  • Обсуждение задач с помощью email.
  • Оценка производительности компании.
  • Отчётность по развитию проекта.
  • Возможность настройки под требования конкретного проекта.
  • Подключение дополнительных модулей (например, статусы задач, учёт времени, этапы проекта, оценка сложности и пр.)
  • Неограниченное количество проектов и места в облачном хранилище.

Преимущество «Планиро» в его универсальности — система построена по принципу, когда в проекте в любое время можно подключить модули с дополнительной функциональностью. Поэтому сервис послужит удобным инструментом и для продвинутых менеджеров, и для тех, кому не надо ничего лишнего. «Планиро» интегрирован с календарём Google, имеет разделение по уровням доступа, мобильную версию и не ограничивает количество проектов для одной организации.

Цены:

  • Пробный период.
  • Имеется 14-дневный тестовый период. Cервис стоит от 240 руб. за сотрудника в месяц или по 8 руб. в день.

Платформы: Web, iOS, Android, Mac

Kanban Tool

Kanban Tool — это веб-инструмент для большей чёткости и наглядности прогресса проекта. Пользователь может воспользоваться цветовыми фишками канбан-доски, установкой ролей, тегами и сроками выполнения работы. С помощью этого сервиса можно постоянно анализировать и улучшать свой рабочий процесс и эффективность бизнеса. Ключевые особенности Kanban Tool:

  • Онлайн канбан-доска.
  • Аналитика.
  • Визуальное управление проектами.
  • Онлайн-документы.
  • Управление задачами через Drag & Drop.
  • Списки дел.
  • Тайм-менеджмент и отчёты.
  • Совместная работа.

Цены:

  • Пробный период.
  • Имеется бесплатный тариф для 2 досок и 2 пользователей, и платные, от $5 за пользователя в месяц.

Способ оплаты: По подписке
